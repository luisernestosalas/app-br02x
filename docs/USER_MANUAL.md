# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar clientes y sus notas. Puedes agregar información de clientes, escribir notas sobre ellos y mantener todo organizado en un solo lugar.

## Cómo acceder

**Dirección web:** https://app-br02x-ewjr1g0xj-luisernestosalas-2775s-projects.vercel.app

Abre esta dirección en tu navegador web (Chrome, Firefox, Safari, etc.).

## Funciones principales

### 1. Ver lista de clientes
- Al abrir la app, verás todos los clientes registrados
- Cada cliente muestra su información básica

### 2. Agregar un nuevo cliente
1. Busca el botón "Agregar Cliente" o "+"
2. Completa los datos solicitados:
   - Nombre
   - Teléfono
   - Email
   - Empresa (opcional)
3. Haz clic en "Guardar"

### 3. Ver detalles de un cliente
1. Haz clic sobre el nombre del cliente
2. Se abrirá una ventana con toda su información
3. Aquí también verás todas las notas asociadas
