# Documentación Técnica - CRM App

## Información General
- **Aplicación**: Sistema CRM para gestión de clientes y notas
- **Stack Tecnológico**: HTML + JavaScript (vanilla)
- **Repositorio**: https://github.com/luisernestosalas/app-br02x
- **URL de Producción**: https://app-br02x-ewjr1g0xj-luisernestosalas-2775s-projects.vercel.app

## Arquitectura del Sistema

### Frontend
- **Framework**: Aplicación web SPA (Single Page Application)
- **Tecnologías Core**:
  - HTML5 para estructura
  - JavaScript vanilla para lógica de negocio
  - CSS para estilos (presumiblemente)

### Hosting y Despliegue
- **Plataforma**: Vercel
- **Tipo**: Aplicación estática
- **Dominio**: Subdominio automático de Vercel

## Funcionalidades Principales

### Gestión de Clientes
- Creación, edición y eliminación de clientes
- Almacenamiento de información de contacto
- Interfaz de usuario para navegación entre clientes

### Sistema de Notas
- Asociación de notas a clientes específicos
- Funcionalidad CRUD para notas
- Persistencia de datos (probablemente localStorage)
