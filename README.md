# CRM Cliente-Notas

Una aplicación web simple y eficiente para gestionar clientes y sus notas asociadas, desarrollada con HTML y JavaScript vanilla.

## 🔗 Enlaces

- **Repositorio:** [https://github.com/luisernestosalas/app-br02x](https://github.com/luisernestosalas/app-br02x)
- **Demo en vivo:** [https://app-br02x-ewjr1g0xj-luisernestosalas-2775s-projects.vercel.app](https://app-br02x-ewjr1g0xj-luisernestosalas-2775s-projects.vercel.app)

## ✨ Características

- 👥 **Gestión de Clientes**: Crear, editar y eliminar clientes
- 📝 **Sistema de Notas**: Añadir y gestionar notas por cliente
- 💾 **Almacenamiento Local**: Los datos se guardan en localStorage del navegador
- 📱 **Diseño Responsivo**: Interfaz adaptable a diferentes dispositivos
- ⚡ **Rendimiento Óptimo**: Aplicación ligera sin dependencias ext
